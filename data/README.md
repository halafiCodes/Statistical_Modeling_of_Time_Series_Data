# Data

## Required Input
Place the Brent oil price dataset at one of:
- data/brent_prices.csv
- data/BrentOilPrices.csv

Expected columns:
- Date (day-month-year or ISO format)
- Price (USD per barrel)

## Optional Outputs
The analysis notebook can export:
- data/change_points.json

Use this file to serve change point results in the dashboard.
